# APLIKASI MANAJEMEN ELPIJI
Aplikasi Desktop Manajemen Elpiji yang dibangun menggunakan library php desktop dan framework codeigniter3.

# FITUR APLIKASI

- Input Tangki
- Gain Tangki
- Laporan
- Sopir
- Agen Elpiji

# JALANKAN APLIKASI

Untuk menjalankan aplikasi :
- Buka Elpiji.exe
- Masukan Username Password admin
- Enjoy Aplikasi
